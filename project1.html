Bandname generator


This Express.js application is a simple web server that uses EJS templating to render dynamic content. The server listens on port 3000 and serves static files from the public directory. It renders an index.ejs file on the root route, which is set up to display a random name generated from predefined arrays of adjectives and nouns when a form is submitted. The application also includes a footer.ejs to dynamically display the current year and uses body-parser to handle form submissions. CSS styling is linked in the header.ejs file to ensure the design is applied to the page.
